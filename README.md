# 自己紹介サイト

シンプルで美しい自己紹介Webページです。

## 特徴

- レスポンシブデザイン対応
- モダンなグラデーション背景
- アニメーション効果付き
- モバイルフレンドリー

## 技術スタック

- HTML5
- CSS3
- グラデーション・アニメーション

## セクション

- **プロフィール**: 名前と職業
- **自己紹介**: 簡単な自己紹介文
- **スキル**: 保有スキル一覧
- **お問い合わせ**: Email、GitHub、Twitter等の連絡先リンク

## 使い方

1. リポジトリをクローン
```bash
git clone https://github.com/katzkawai/Copilot.git
```

2. `index.html`をブラウザで開く

## カスタマイズ

`index.html`を編集して以下を変更できます：
- 名前と職業
- 自己紹介文
- スキル一覧
- 連絡先リンク

`style.css`でデザインをカスタマイズできます。
